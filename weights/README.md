Juggernaut-XL-v9, see [../docs/methodology.md](../docs/methodology.md) for
training configuration) is **not** stored in this git repository — it's too

alongside the code/metadata archive that Zenodo pulls automatically from a
GitHub Release. See the root [README.md](../README.md#releases--dois) for
how that upload works.
