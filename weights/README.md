# Trained weights

`Arynwood_lora.safetensors` (456 MB, SDXL LoRA, trained against
Juggernaut-XL-v9, see [../docs/methodology.md](../docs/methodology.md) for
training configuration) is **not** stored in this git repository — it's too
large for a normal git push and doesn't belong in version control.

It is uploaded directly as an additional file on the Zenodo record for this
project, alongside the code/metadata archive that Zenodo pulls automatically
from a GitHub Release. See the root [README.md](../README.md#publishing-a-new-version)
for the manual upload steps.
